# Javascript: Salvando dados localmente com IndexedDB
Curso Alura

Instalação:
-> .\client\npm install
Descompactar server.zip na pasta raiz

Execução do projeto:
-> .\server\npm start
Abrir localhost:3000

Modificações no projeto:
-> .\client\npm run build
ou para modificações em tempo de execuç
-> .\client\npm run watch
